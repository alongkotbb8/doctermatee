# QA Checklist — ใบกำกับภาษี (Tax Invoice)

> Prerequisite: migration 011 + 012 apply แล้ว ✅ · ต้องมีบัญชี admin + user · Omise test mode
> บัตรทดสอบ: `4242 4242 4242 4242` / 12/30 / 123

---

## 1. ฟอร์ม Checkout

- [ ] ช่อง **ตำบล/แขวง** แสดงแยกจาก เขต/อำเภอ
- [ ] เว้นช่องตำบลว่าง → กดสั่งซื้อ → ช่องไฮไลต์แดง ไม่ผ่าน
- [ ] ติ๊ก **"ขอใบกำกับภาษีแบบเต็มรูป"** → ฟอร์มกาง (ประเภท/ชื่อ/สำนักงาน/เลขภาษี/ที่อยู่/อีเมล/เบอร์)
- [ ] เลือก "นิติบุคคล" → label เป็น "ชื่อบริษัท" / "บุคคลธรรมดา" → "ชื่อ-นามสกุล"
- [ ] เลือก "สำนักงานใหญ่" → ช่องเลขสาขา ล็อกเป็น `00000` / เลือก "สาขา" → กรอกเลขสาขาได้
- [ ] กรอกเลขผู้เสียภาษีไม่ครบ 13 หลัก → error "เลขประจำตัวผู้เสียภาษี 13 หลัก"
- [ ] ไม่ติ๊กขอเต็มรูป → สั่งซื้อได้ปกติ (จะออกใบย่อให้)

---

## 2. ออกเลขอัตโนมัติ (ตอนชำระเงิน)

- [ ] สั่งซื้อ **ไม่ติ๊ก** ขอเต็มรูป → จ่ายเงินสำเร็จ → เกิดใบ **`CD2026MM0001`** (MM=เดือนปัจจุบัน)
- [ ] สั่งซื้อ **ติ๊ก** ขอเต็มรูป → จ่ายเงินสำเร็จ → เกิดใบ **`ICD2026MM0001`** (ไม่มี CD คู่)
- [ ] สั่งหลายออเดอร์ติดกัน → เลขวิ่ง 0001 → 0002 → 0003 ไม่ซ้ำ ไม่ข้าม
- [ ] (ถ้าทดสอบได้) ขึ้นเดือนใหม่ → เลขรีเซ็ตเป็น 0001

---

## 3. แท็ก + รายการออเดอร์ (`/admin/orders`)

- [ ] ออเดอร์ที่ขอเต็มรูป → มีแท็ก **🧾 ขอใบกำกับเต็มรูป** (ให้คนแพ็ครู้)
- [ ] ออเดอร์ที่ชำระแล้ว → มีลิงก์ **🧾 ใบกำกับ**
- [ ] ออเดอร์ยังไม่ชำระ → ไม่มีลิงก์ใบกำกับ

---

## 4. หน้าพิมพ์ (`/admin/orders/[id]/tax-invoice`)

- [ ] แสดง **2 ชุด: ต้นฉบับ (Original) + สำเนา (Copy)**
- [ ] ข้อมูลผู้ขาย = บริษัท แคร์ดีแม็กซ์ จำกัด + เลขภาษี 0135560013047
- [ ] **ใบย่อ (CD):** หัวข้อ "ใบกำกับภาษีอย่างย่อ" · ลูกค้า+ที่อยู่ = จากที่อยู่จัดส่ง
- [ ] **ใบเต็ม (ICD):** หัวข้อ "ใบกำกับภาษี" · ชื่อบริษัท + เลขผู้เสียภาษี + สำนักงานใหญ่/สาขา + ที่อยู่ที่กรอก
- [ ] รายการสินค้าครบ (ชื่อ/จำนวน/ราคา/มูลค่า)
- [ ] **VAT ถูกต้อง:** มูลค่าที่คำนวณภาษี + VAT 7% = รวมทั้งสิ้น (= ยอดจ่ายจริง)
  > ตรวจเลข: ถ้ารวม ฿854 → ฐาน ฿798.13 + VAT ฿55.87 = ฿854 ✓
- [ ] กดปุ่ม "🖨️ พิมพ์" → preview แยกเป็น 2 หน้า (ต้นฉบับ/สำเนา คนละหน้า) ปุ่มไม่ติดมาในงานพิมพ์
- [ ] เปิดใบของออเดอร์ที่ยังไม่จ่าย → ขึ้น "ยังไม่มีใบกำกับภาษี"

---

## 5. Export Excel (`/admin/tax-invoices`)

- [ ] หน้าแสดงรายการใบกำกับภาษีทั้งหมด (เลขที่/วันที่/ประเภท/ผู้ซื้อ/ฐาน/VAT/รวม)
- [ ] เลือกช่วงวันที่ → กด **⬇ Export Excel** → ได้ไฟล์ `.xlsx`
- [ ] เปิดไฟล์: คอลัมน์ครบ — เลขที่ / วันที่ / ประเภท / เลขออเดอร์ / ชื่อผู้ซื้อ / เลขผู้เสียภาษี / มูลค่าก่อน VAT / VAT 7% / รวมทั้งสิ้น
- [ ] ตัวเลขฐาน + VAT + รวม ตรงกับใบที่ออก ส่งบัญชีได้

---

## 6. สิทธิ์ / ความปลอดภัย

- [ ] เปิด `/admin/tax-invoices` หรือ `/api/admin/tax-invoices/export` โดยไม่ใช่ admin → ถูกปฏิเสธ (403/redirect)
- [ ] หน้าร้าน/ลูกค้า มองไม่เห็นใบกำกับภาษีของคนอื่น (RLS admin-only)

---

## ✅ พฤติกรรมที่ตั้งใจ (ไม่ใช่บั๊ก)

- ใบกำกับภาษีออก **เฉพาะตอนชำระเงินแล้ว** — ออเดอร์ unpaid ยังไม่มีใบ
- **ใบย่อไม่มีเลขผู้เสียภาษีผู้ซื้อ** — ปกติของใบกำกับภาษีอย่างย่อ
- 1 ออเดอร์ = 1 ใบ (ขอเต็ม=ICD เท่านั้น, ไม่ขอ=CD) — ไม่ออกซ้ำ
- ในตาราง counter อาจเห็น `TEST/209901` — มาจากการทดสอบฟังก์ชัน ไม่กระทบเลขจริง CD/ICD
